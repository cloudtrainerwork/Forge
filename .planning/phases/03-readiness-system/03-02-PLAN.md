## Goal

Build interactive frontend components for visualizing and updating 6-dimensional readiness on graph nodes, with support for both percentage and state views, color coding, and direct click interactions.

## Context

From user requirements:
- Click directly on nodes to open readiness forms
- Visual indicators overlay on nodes (not disruptive to layout)
- Support both percentage (0-100%) and state (3+ states) display modes
- Color coding for status (red="on the bubble", yellow=in-progress, green=complete)
- Handle screen-based deliverables with all 6 dimensions
- Support for grouping and bulk operations
